# CSV Cleaner & Reporter (Automation Script)

A small automation tool that takes messy CSV/Excel exports and produces:
- a cleaned CSV file (consistent columns, trimmed text, fixed dates/numbers)
- a validation report (missing values, duplicates, invalid rows)

This is useful for:
- importing data into databases/CRMs
- cleaning exports from accounting systems, websites, or legacy apps
- creating a repeatable data pipeline for small businesses

## Features
- Supports CSV and Excel (.xlsx)
- Normalizes column names
- Trims whitespace and removes empty rows
- Validates required columns
- Generates a summary report
